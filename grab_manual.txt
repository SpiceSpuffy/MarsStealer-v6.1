Грабер файлов (Grab) - собирает файлы в системе по заданным параметрам

Создание правила:
Name - желаемое название архива с файлами в логе (к примеру, если указать Desktop, то архив будет называться Desktop.zip)
Max Size - максимальное значение веса архива в килобайтах. 0 - не ограничено по весу.
Path - стартовый путь для сбора файлов, доступны следующие константы:
%DESKTOP% - рабочий стол пользователя
%APPDATA% - Путь к папке Roaming (C:\Users\*user*\AppData\Roaming)
%LOCALAPPDATA% - Путь к папке Local (C:\Users\*user*\AppData\Local)
%USERPROFILE% - Путь к папке пользователя (C:\Users\*user*\)
Formats - список масок для сбора файлов. Поддерживаются маски вида *filename*.*format*, к примеру:
*.txt - будет собирать все файлы с форматом .txt
*2fa*.* - будет собирать все файлы с любым форматом, в названии которого содержится 2fa
test*.* - будет собирать все файлы, название которых начинается на test
*test.* - будет собирать все файлы, название которых заканчивается на test
Маски можно комбинировать в любом удобном для вас виде, чтобы максимально настроиться на нужные файлы
Collect recursively - рекурсивно или нет работать правилу. Рекурсивно - будет искать по всем папкам, которые есть в Path. Если же не включать рекурсию, то соберет только подходящие файлы по маскам в указанной папке.
